First steps with notebooks
##########################

The Rubin Science Platform's Notebook Aspect is based on JupyterLab, and enables you to create and run interactive Jupyter notebooks and use a web-based shell for an immersive data analysis environment that's hosted next to Rubin data.
In this quick-start tutorial, you will learn how to start a notebook server and become familar with the Notebook Aspect.
